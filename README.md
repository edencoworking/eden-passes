# Eden Passes (Frontend)

A React frontend for managing coworking passes and customers.

*Backend code removed in v1.2.0; a mock/data layer will be introduced in a subsequent PR.*

## Development

Install:
```bash
npm install
```

Run frontend:
```bash
npm start
```

Build for production:
```bash
npm run build
```

Run tests:
```bash
npm test
```
